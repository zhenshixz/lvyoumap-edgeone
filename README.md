# 🗺️ 中国旅游智能导览地图 (Android Tablet Adapted Version)

## EdgeOne 稳定部署版

本仓库已调整为 **静态优先** 的 EdgeOne Makers/Pages 部署结构：

- `backend/db.json` 仅作为本地数据维护源，不进入线上构建产物。
- 构建时自动生成 `data/provinces/*.json` 和 `data/search-index.json`，避免任何单文件超过 EdgeOne 的 25 MiB 限制。
- 线上不依赖 Express、可写文件系统、第三方天气接口或伪云数据库。
- 收藏保存在当前浏览器的 `localStorage` 中；清理浏览器数据会一并清除收藏。
- `scripts/build_edgeone.js` 会在发布前检查单文件大小和文件总数，超限时给出明确错误。

### Git 仓库自动部署

EdgeOne 会读取根目录的 `edgeone.json`，使用以下固定配置：

- 安装命令：`npm install --no-audit --no-fund`
- 构建命令：`npm run build`
- 输出目录：`dist`
- Node.js：`20.18.0`

将本次修改提交并推送到 EdgeOne 已连接的分支后，重新部署即可。不要把输出目录改回项目根目录，否则本地维护用的 `backend/db.json` 会再次被纳入部署检查。

### 本机预览

双击 `preview_edgeone.bat`。脚本会检查 Node.js 版本、生成与 EdgeOne 完全相同的 `dist` 产物，再启动 `http://localhost:8080`，同时显示局域网访问地址。

也可以在终端运行：

```powershell
npm run build
npm run preview
```

### 直接上传

先运行 `npm run build`，然后在 EdgeOne 中上传 `dist` 文件夹或把 `dist` 内的内容压缩为 ZIP 后上传。`index.html` 必须位于上传内容最外层。

这是一个专为**安卓平板（Android Tablet）**横屏体验适配的、基于 HTML5 / CSS3 / JavaScript 和 Apache ECharts 的中国动态旅游导览地图。

与传统网页不同，本项目特别进行了**平板触控交互优化**与**离线运行优化**。

## 🚀 特色功能

1. **平板端专享布局**：
   - **双栏非滚动视口**：左侧 55% 动态地图，右侧 45% 信息抽屉，配合 `100vh` 高度，带来原生 App 般的使用体验。
   - **大尺寸触控靶区**：控制按钮尺寸均大于 `44px`，卡片间距适宜，无悬浮态粘连，全面优化手势触控。

2. **多模式动态中国地图**：
   - **省份视图**：点击省份区块展示该省份的所有旅游景点。
   - **热门城市视图**：在地图上渲染出发光的**波动涟漪点（EffectScatter）**，点击可直接穿透进入城市（如成都、西安等）。
   - **自由缩放/平移**：支持双指缩放、拖拽，并提供重置与缩放按钮。

3. **智能数据处理**：
   - **口碑推荐排序（默认）**：景点默认按照真实的游客口碑评分（Rating）从高到低排列。
   - **灵活排序**：支持按照“人气热度”、“门票价格升序”、“门票价格降序”进行实时重新排序。
   - **全局智能搜索**：支持在顶部输入框搜索省份、城市、景点级别（如“AAAAA”）、景点关键字等。
   - **离线与 CORS 免配置**：将 4.2MB 的 GeoJSON 地理边界数据编译为了 `china_geo.js` 文件，支持直接在平板上双击 `index.html` 离线使用（`file://` 协议），不会触发浏览器的跨域 CORS 拦截。

4. **高级视觉与微交互**：
   - **毛玻璃与暗黑风（Glassmorphism）**：科技与文化交融的深色背景配上高亮青色/金色发光质感。
   - **系统气象联动**：右侧面板的天气信息将与顶部状态栏及城市实时气象状态深度关联，并模拟实时时钟。
   - **多页签详情弹窗**：点击景点卡片展示大图，并设有“基本信息”（带口碑评分条滑动动画）、“游玩路线图（时间轴）”和“精选游客点评”。
   - **本地收藏夹**：支持景点收藏/取消，数据持久化保存在本地 `localStorage`，重新打开也不会丢失。

## 📂 项目结构

```
map/
├── index.html       # 网站主入口，结构声明与矢量图标 SVG 嵌入
├── style.css        # 精美的平板端玻璃拟态样式表与过渡动画

└── README.md        # 本使用指南说明文档
```

## 📱 平板端部署与使用指南

### 方法一：极速本地离线使用 (推荐 ⭐️)
1. 将 `map` 文件夹整体传输或同步到您的安卓平板上。
2. 在安卓平板上使用任意浏览器（如 Chrome、Edge 或 Kiwi Browser）的**文件管理/浏览器打开**功能，直接打开 `index.html`。
3. 即可在完全没有网络连接的情况下自由流畅体验！

### 方法二：局域网在线预览
若您在电脑上进行调试，想用平板实时预览：
1. 在本工程的 `d:\anti` 根目录运行一个简易网页服务器。例如，使用 Python：
   ```bash
   python -m http.server 8000
   ```
2. 确保您的平板和电脑处于同一个 Wi-Fi 网络下。
3. 在平板浏览器中输入电脑的 `IP地址:8000/map/index.html`，即可进行体验。
